About Me

Enjoying the northwest, staying close to nature, and finding joy in tech work, I moved to Seattle many years ago. I've worked as an applications engineer after graduating, focusing on Azure cloud services and legacy migrations with Windows Server Enterprise, IIS Manager, and SQL Servers software. Supporting Azure cloud services and Windows based enterprise applications and online websites, I specialize in web app and virtual machine environment setups, live-site quick fixes and on-call escalated support, to longterm maintenance, monitoring, and on-premises system updates and rebuilds to the newer cloud platforms. After spending the last year studying Agile Methodology and PMI, this year my university studies are from the development perspective, building websites using front-end and server-side technologies - VSCode, Gitbash, JavaScript, jQuery, HTML, CSS, Node.js - and uploading to Github and in some cases connecting with third party APIs.
